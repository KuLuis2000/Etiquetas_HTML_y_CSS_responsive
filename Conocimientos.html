Conocimientos

  • Conococimientos avazandos de HTML5:
    HTML5 es la última versión del lenguaje de marcado estándar para crear páginas web. Un conocimiento avanzado de HTML5 implica comprender y utilizar todas sus características, como elementos semánticos (por ejemplo, < header >, < nav >, < article >), formularios avanzados, APIs de audio y video, elementos de lienzo (< canvas >), almacenamiento local (localStorage y sessionStorage), entre otros. Esto permite crear estructuras web modernas y eficientes.

  • Conocimientos avanzados de CSS3:
    CSS3 es el estándar para la presentación y el diseño de páginas web. Un conocimiento avanzado implica dominar propiedades avanzadas como animaciones, transiciones, transformaciones, flexbox y grid layouts, selección de pseudo-elementos, sombras y gradientes, entre otros. Estos conocimientos permiten diseñar sitios web atractivos y responsivos.

  • Conocimientos avanzados del lenguaje Javascript:
    JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web. Un conocimiento avanzado implica comprender conceptos como funciones avanzadas, closures, programación orientada a objetos, promesas, asíncrono y manejo de errores. Además, es importante estar al tanto de las últimas características y actualizaciones del lenguaje.

  • Implementación de interacciones con Javascript y consumo de datos con API's:
    Este punto se refiere a la capacidad de utilizar JavaScript para crear interacciones dinámicas en una página web, como formularios con validación en tiempo real, despliegue de contenido dinámico y la comunicación con servidores a través de API's (Application Programming Interfaces) para obtener y enviar datos. Esto implica comprender cómo hacer solicitudes HTTP, manejar respuestas de API y actualizar el DOM de manera dinámica.

  • Manejo básico de Git:
    Git es un sistema de control de versiones ampliamente utilizado en el desarrollo de software. Un manejo básico implica saber cómo inicializar un repositorio, realizar confirmaciones (commits), crear y fusionar ramas, clonar repositorios y colaborar en proyectos utilizando plataformas como GitHub o GitLab.

  • Implementación de los archivos en el servidor web (DEPLOY):
    Desplegar un sitio web implica subir los archivos y recursos necesarios a un servidor web para que estén disponibles en línea. Esto puede implicar el uso de protocolos de transferencia de archivos (FTP), servicios de alojamiento web, servidores en la nube (como AWS o Heroku), o sistemas de contenedores como Docker. Es importante comprender cómo configurar y administrar el entorno de despliegue de un sitio web para que funcione de manera eficiente y segura.
